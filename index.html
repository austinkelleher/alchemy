Alchemy is a utility and contrib extension to the base TensorFlow library. It includes notebooks and tests for recent research papers and my own experiments.

Getting started¶

Installing Alchemy requires pip. After that, you’re set.

git clone https://github.com/wenkesj/alchemy
(cd alchemy; pip install -e .)

Check out these notebooks that include detailed examples and implementations of recent research and experiments. All notebooks were created with Google Colab and can be opened and edited freely!:
