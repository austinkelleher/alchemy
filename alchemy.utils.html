alchemy.utils.distribution_utils.custom_scale_mixture_prior_factory(pi=0.01, sigma1=0.25, sigma2=0.01)¶

Creates a prior builder and a posterior builder from the pi, sigma1 and sigma2.

Example usage:

``` import sonnet.python.custom_getters.bayes_by_backprop as bbb

prior_builder, posterior_builder = custom_scale_mixture_prior_factory(
pi=.01, sigma1=.25, sigma2=.01)

sampling_mode_ph = tf.placeholder(tf.string, []) get_bbb_variable_fn = bbb.bayes_by_backprop_getter(

posterior_builder=posterior_builder, prior_builder=prior_builder, kl_builder=bbb.stochastic_kl_builder, sampling_mode_tensor=sampling_mode_ph)

```
